======================================
      Claude Code 新手入门指南
======================================

一、什么是Claude Code？
-------------------
Claude Code是Anthropic公司开发的官方命令行工具（CLI），让你可以通过终端与Claude AI进行交互，完成各种编程任务。

二、安装方法
----------
1. 通过VS Code安装（推荐）：
   - 打开VS Code
   - 在扩展市场搜索"Claude Code"
   - 点击安装

2. 通过命令行安装：
   - 访问 https://github.com/anthropics/claude-code
   - 按照官方文档指引安装

三、基本使用
----------
1. 启动Claude Code：
   - 在VS Code中按 Ctrl+Shift+P (或 Cmd+Shift+P)
   - 输入"Claude Code"并选择启动
   - 或直接在终端输入：claude

2. 与Claude对话：
   - 直接用中文或英文描述你的需求
   - 例如："帮我写一个Python函数计算斐波那契数列"

四、常用功能说明
--------------

1. 写代码
   - 描述需求："写一个Hello World程序"
   - Claude会自动创建或修改文件

2. 修改代码
   - "把变量名从count改成num"
   - "给这个函数添加注释"

3. 解释代码
   - "帮我解释这段代码是什么意思"
   - "这个函数的作用是什么"

4. 调试代码
   - "我的程序报错了，帮我看看"
   - 把错误信息贴给Claude

5. 运行命令
   - "运行这个Python文件"
   - "安装npm依赖"

五、实用技巧
----------

1. 斜杠命令（快捷指令）
   /help          - 查看所有可用命令
   /clear         - 清空对话历史
   /commit        - 创建Git提交
   /diff          - 查看代码差异

2. 提问技巧
   - 清晰描述你的目标
   - 说明你使用的编程语言
   - 提供相关代码片段
   - 明确说出遇到的问题

   好的问题示例：
   "帮我在Python中写一个读取CSV文件并打印前10行的函数"

   不好的问题示例：
   "怎么写代码"

3. 文件操作
   Claude可以帮你：
   - 创建新文件
   - 读取现有文件
   - 修改文件内容
   - 删除文件

六、常见问题解答
--------------

Q: Claude能理解中文吗？
A: 可以！Claude支持中文对话。

Q: 如何让Claude修改特定文件？
A: 告诉Claude文件名或路径即可，如："修改test.py文件，添加错误处理"

Q: Claude能执行命令吗？
A: 可以，Claude可以运行bash命令、Python脚本等

Q: 我的代码很长，Claude能处理吗？
A: 可以，Claude可以处理大型代码库

七、注意事项
----------
1. 首次使用需要登录Anthropic账号
2. 使用API可能产生费用（取决于你的订阅）
3. 重要的代码记得备份
4. 不要让Claude直接删除重要文件

八、进阶使用（了解即可）
---------------------
1. 配置自定义提示词
2. 使用工作区(workspace)
3. 集成到其他开发流程
4. 创建自定义技能(skill)

九、练习建议
----------
作为0基础新手，建议从以下练习开始：

1. 基础练习：
   - "帮我在当前文件夹创建一个hello.txt文件，内容是Hello World"

2. 编程练习：
   - "用Python写一个计算器程序"
   - "帮我写一个简单的网页"

3. 调试练习：
   - 写一个有bug的小程序，让Claude帮你找错误

十、总结
-------
Claude Code是你编程路上的好帮手！
- 大胆提问
- 清晰描述需求
- 多练习，多交流

遇到问题随时问Claude，它会耐心解答。

祝你学习愉快！
